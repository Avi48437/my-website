Sub-Gaussian Random Variables

Motivation#

We work on the real line $\R$. Expectation and probability are denoted by $\Eb$ and $\Pb$.

Sub-Gaussian random variables play a central role in high-dimensional probability.

Definition (Sub-Gaussian)
A random variable $X$ is called sub-Gaussian if there exists $\sigma>0$ such that $$ \Eb[\exp(tX)] \le \exp\left(\frac{\sigma^2 t^2}{2}\right), \quad t \in \R \tag{1} $$
Lemma (Tail Bound)

If $X$ is sub-Gaussian with parameter $\sigma$, then for all $u > 0$,

$$ \Pb(|X|\ge u)\le 2\exp\left(-\frac{u^2}{2\sigma^2}\right)\tag{2} $$

Proof.
Apply Chernoff’s inequality and optimize over $t>0$.
□

R illustration#

We simulate a standard Gaussian variable and compare empirical and theoretical tails.

Code 
set.seed(1)

n <- 10^5
x <- rnorm(n)   # standard Gaussian (sub-Gaussian)

u <- seq(0, 4, length.out = 50)
emp_tail <- sapply(u, function(t) mean(abs(x) >= t))
theory_tail <- 2 * exp(-u^2 / 2)

head(data.frame(u, empirical = emp_tail, theory = theory_tail))

Sub-Gaussian variables are fundamental in high-dimensional probability (Vershynin 2018).

Vershynin (2018) shows that sub-Gaussian variables have Gaussian-like tails.
